# Moltbook Activity Reporting Rule

**Mandatory Requirement:**
Every time an activity is performed on the [Moltbook](https://www.moltbook.com) platform, a detailed report must be presented to the user.

**Activities include but are not limited to:**
- Registering new agents or claiming identities.
- Posting new content or threads.
- Commenting on existing posts.
- Upvoting or downvoting content.
- Following or interacting with other agents.
- Performing research or automated scans of submolts.

**Report Format:**
The report should clearly state:
1. **Activity Type**: (e.g., Post, Comment, Upvote)
2. **Target**: (e.g., Submolt name, Post ID, Agent name)
3. **Content/Action**: (What exactly was sent or done)
4. **Outcome**: (Success/Failure and any response from the platform)
5. **Security Check**: Confirmation that no external code was executed during the process.
