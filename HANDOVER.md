# Project Handover: Agent Constitution (Universal Agentic Structure)

## 🎯 Project Overview
**Agent Constitution** is a high-standard, modular context architecture designed for AI Agents (Antigravity, Cline, Windsurf, Jules). The project's mission is to eliminate AI "spaghetti code" and hallucinations by forcing agents to strictly follow a defined "Project Constitution" (UAS - Universal Agentic Structure).

It operates on the principle of **Modular Context**: instead of one massive prompt, agents load specific **Rules**, **Workflows**, and **Skills** only when needed.

---

## 🏗️ Core Architecture
The project is structured to support both international (English) and local (Persian) agentic workflows:

- **`.agent/rules/`**: Global system rules in professional English (Coding, TDD, Git, Security).
- **`fa/rules/`**: Persian translation of Global rules for specialized Persian-speaking agents.
- **`.agent/skills/`**: A library of 64+ technical skills (FFmpeg, Python, DevOps, AI, etc.) with standardized metadata.
- **`fa/skills/`**: Persian localized technical skills.
- **`.agent/workflows/`**: Strict operational roadmaps (Init, Documentation, QA, AI Optimization).
- **`memory-bank/`**: Cross-session state management (Context, Cost Tracking).
- **`.clinerules`**: The "bootloader" that tells VS Code agents where to find their instructions.
- **`AGENTS.md`**: The master manual for any AI agent entering this repository.
