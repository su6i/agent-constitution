# قوانین اصلی

## کنترل هزینه (۲۴/۷ CRITICAL)
- قبل از هر تسک بزرگ، پیچیدگی آن را اعلام کن: TRIVIAL / MODERATE / CRITICAL
- اگر TRIVIAL است: فقط کد بنویس بدون توضیح اضافه.
- اگر مطمئن نیستی: توقف کن و فقط یک سوال بپرس.
- بیش از ۲۰ فایل را در یک جلسه بدون تایید ویرایش نکن.

## ساختار پاسخ
- پاسخ‌ها را با کد شروع کن.
- از فرمت‌های استاندارد [✅ done] استفاده کن.

- زبان: Python 3.12+
- مدیریت پکیج: فقط `uv`.
- ورودی اصلی: `main.py`.
