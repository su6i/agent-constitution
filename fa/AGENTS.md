# AGENTS.md — راهنمای agent برای این پروژه (نسخه فارسی)

## خلاصه پروژه
این مخزن شامل قوانین و استانداردهای برنامه‌نویسی ایجنتیک ۲۰۲۶ است که بر توازن هزینه و قدرت پردازشی تمرکز دارد.

## اسناد مرجع
- سند فنی شاخص فارسی: `../AGENTIC-CODING-SETUP.fa.md`
- ورودی فارسی مخزن: `../README.fa.md`
- قرارداد اجرایی عمومی: `../AGENTS.md`

## ساختار فارسی
- `fa/skills/` — راهنماها و مهارت‌های هوشمند به زبان فارسی.
- `fa/memory-bank/` — قالب‌های کانتکست جلسه.
- `fa/workflows/` — دستورالعمل‌های اجرایی (SOP).
- `fa/rules/` — ترجمه قوانین `.agent/rules` (نسخه محلی).

## قوانین مهم برای Jules و سایر ایجنت‌ها
- تمام تغییرات در شاخه فارسی (`fa/`) باید با استانداردهای فایل انگلیسی مطابقت داشته باشد.
- اگر فایل اصلی انگلیسی به ریشه promote شده، نسخه فارسی آن نیز باید در مسیر canonical فارسی sync شود.
- از `uv` برای مدیریت پکیج‌ها استفاده کنید.
- فرمت کامیت: `[نوع]: [توضیح]` (مثلاً `feat: اضافه کردن مهارت جدید`).

## چطور تست اجرا کنید

uv run pytest --cov=src
