# شروع session کدنویسی

## قبل از شروع — agent این را بخوان

1. فایل `memory-bank/activeContext.md` را بخوان
2. فایل `memory-bank/progress.md` را بخوان
3. وضعیت فعلی را خلاصه کن در یک پاراگراف
4. task بعدی را از `memory-bank/progress.md` بردار
5. پیچیدگی task را اعلام کن: TRIVIAL / MODERATE / CRITICAL
6. مدل مناسب را پیشنهاد بده:
   - TRIVIAL → DeepSeek V3.2
   - MODERATE → MiniMax M2.5
   - CRITICAL → Claude Sonnet 4.6

## بعد از هر task
1. `memory-bank/activeContext.md` را آپدیت کن
2. هزینه تقریبی را در `memory-bank/costLog.md` ثبت کن
3. commit با format درست بزن
4. تایید بگیر برای task بعدی
